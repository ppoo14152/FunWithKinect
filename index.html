Descripción de proyecto

FunWithKinect

Consiste en un vídeo juego para computadora manejado por kinect, se encuentra conformado por una serie de minijuegos, con los cuales podrás divertirte y ejercitarte.

  • Video de Proyecto

Descripción de Interfaz

Menus

La pantalla principal del vídeo juego nos da 3 opciones:

  • 1-. Juegos: Despliega otro menú con la lista de minijuegos, los cuales podremos seleccionar.
  • 2-. Récords: Nos muestra las puntuaciones mas altas.
  • 3-. Ayuda: Nos explica como jugar los diferentes minijuegos (movimientos y objetivos) debido a que cada minijuego tiene una forma diferente de jugarse.

Cursor-------------------------

Podrás desplazar el cursor por la pantalla solo moviendo tu mano derecha en la dirección que desees y situarlo sobre la opción que quieres seleccionar. El cursor cargara automáticamente la opción."

Requisitos para jugar -----------

Windows

FunWhitKinect funciona en la plataforma de Windows(xp,8,7,vista), requiere la instalación de los programas Geeenfoot kinect Server Y la librería OpenNI y el controlador de OpenNI, los cuales pueden descargarse de los siguientes links:

  • Kinect Server
  • OpenNi
  • controlador de openNI

Una vez instalados los programas se tiene que ejecutar el server antes de comenzar a jugar, puede hacerse desde menú de inicio->buscar y escribimos Greenfoot Kinect Server damos doble clik y listo.

Si usted requiere usarlo en alguna otra plataforma siga los pasos mencionados en el siguiente link:

Linux

Paquetes necesarios:

  1. sudo apt-get install g++ libglut3-dev libboost-all-dev
  2. sudo apt-get install libwxbase2.8-dev libwxgtk2.8-dev wx-common

Instalar :

  • openNI:
  • mkdir OpenNI
  • cd OpenNI
  • wget http://www.greenfoot.org/doc/kinect/OpenNI-Linux32.tar.bz2
  • tar -jxf OpenNI-Linux32.tar.bz2
  • sudo ./install.sh
  • cd ..

Instalar :

  • NITE:
  • mkdir NITE
  • cd NITE
  • wget http://www.greenfoot.org/doc/kinect/NITE-Linux32.tar.bz2
  • tar -jxf NITE-Linux32.tar.bz2
  • echo '0KOIk2JeIBYClPWVnMoRKn5cdY4=' | sudo ./install.sh
  • cd ..

Instalar :

  • KinectDriver:
  • mkdir Kinect
  • cd Kinect
  • wget http://www.greenfoot.org/doc/kinect/SensorKinect-Linux32.tar.bz2
  • tar -jxf SensorKinect-Linux32.tar.bz2
  • sudo ./install.sh
  • cd ..
